# Totara Notepad

It is a simple text editor, which I designed for one assignment during my graduate diploma study in Auckland.


# Features

  - Open a .txt file
  - Edit a .txt file
  - Save a .txt file
  - Create a .txt file 
  - Change font size
  - Change font family
  - Change font color
